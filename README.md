# 🐼 拼音乐园 - 幼小衔接趣味拼音学习 App

> 面向 3-8 岁幼儿，趣味性、互动性、奖励驱动的跨平台拼音学习应用

---

## 📱 平台策略

| 平台 | 框架 | 最低版本 |
|------|------|---------|
| Android | Kotlin + Jetpack Compose | Android 6.0 (API 23) |
| iOS | Swift + SwiftUI | iOS 14.0 |
| 跨平台共享层 | React Native (可选方案) | - |

**推荐方案：原生双端开发**（Android Kotlin + iOS Swift），保证最佳性能和儿童级流畅体验。

## 🎯 核心功能模块

### 模块1：拼音学习殿堂
- **声母学习**（21个）：b p m f / d t n l / g k h / j q x / zh ch sh r / z c s / y w
- **韵母学习**（35个）：单韵母、复韵母、前后鼻韵母
- **声调学习**（4声 + 轻声）
- **整体认读**（16个）
- 每个知识点：动画演示 + 真人标准发音 + 口型动画

### 模块2：趣味互动小游戏（6款）
1. **拼音配对游戏** - 连线匹配声母+韵母
2. **打地鼠认读** - 看到拼音快速点击对应文字
3. **跟我读** - 语音识别模仿练习
4. **拼音积木** - 拖拽组合拼音
5. **音节接龙** - 同韵母接龙挑战
6. **迷宫探险** - 按拼音指示走迷宫

### 模块3：课后巩固知识点
- **每日5题** 智能出题
- **错题本** 自动收录
- **周测挑战** 阶段性测验
- **知识地图** 可视化进度

### 模块4：奖励激励系统
- **星星系统**：每学完一个知识点获得1-3颗星
- **徽章成就**：20+ 个成就徽章（第一次发音、连续7天等）
- **宠物养成**：学习积分兑换喂食、装扮宠物
- **勋章墙**：展示所有已获成就
- **进度树**：拼音知识树形结构可视化解锁

## ⭐ 奖励系统设计

```
学习 → 获得星星 → 积分
    → 解锁徽章 → 成就系统
    → 喂养宠物 → 宠物成长
    → 开放新章节 → 持续激励
```

| 行为 | 奖励 |
|------|------|
| 完成一个字母学习 | ⭐ 1颗星 |
| 首次发音成功 | 🏅 特殊徽章 |
| 游戏通关 | 🍎 宠物食物 ×3 |
| 连续学习7天 | 👑 坚持勋章 |
| 本周测试满分 | 🎉 动画庆祝 + 大礼包 |

## 🎮 游戏化设计原则

1. **即时反馈**：答对立即播放欢呼音效+动画，答错温和提示不打击
2. **难度渐进**：自动适应学习进度，不过难也不太简单
3. **短时任务**：每个环节控制在 3-5 分钟内（儿童注意力特点）
4. **父母看板**：家长可查看孩子学习报告和进度

---

## 🛡️ 防沉迷系统

为保护幼儿健康，游戏集成防沉迷验证：

| 功能 | 说明 |
|------|------|
| 每日限制 | 30分钟游戏时长 |
| 会话限制 | 单次15分钟强制休息提醒 |
| 汉字验证 | 4位数字转大写汉字点选验证 |
| 验证触发 | 每3局游戏验证一次 |

**验证码示例**：数字 `1234` → 显示 `壹贰叁肆`

## 📲 Android 安装指南

### 方式一：使用 Android Studio
1. 打开 Android Studio → `File` → `Open` → 选择 `android` 文件夹
2. 等待 Gradle 同步完成
3. `Build` → `Build APK(s)` → `Build APK`
4. 将 APK 安装到手机或通过 `adb install` 安装

### 方式二：使用构建脚本
```powershell
cd android
.\build.bat
```

详细说明见 [安装指南](docs/INSTALL.md)
